一、类(Class)

        Object-C为每个类都创建了两个文件，分别为.h(头文件)和.m(实现文件)。一般情况下，.h文件中声明属性和方法。在.m文件中实现在.h文件中声明的方法。有时候为了隐藏实现功能的细节，可以不在.h文件中声明方法或属性，直接在.m文件中实现所需功能，这体现了面向对象的封装特性。

1.1 类的定义与实现

        当我们使用Xcode创建一个类的时候，可以使用快捷键Command + N,Xcode会自动创建.h和.m两个文件

1.2 类的继承

类的继承主要有两种方式：

  • 在Xcode中，依次点击File -> New -> File -> Cocoa Touch Class，即可新增类，在Subclass of中输入或者选择新增类的父类，如下图所示：新增的类是NSObject类的子类。

  • 直接通过代码来实现继承(ClassOne继承自ClassTwo)

        被继承的类称为父类（基类、超类），继承的类称为子类。子类拥有父类所有的方法和属性(除了私有的)，所以在继承后的子类可以使用从父类继承而来的方法和属性
