Markdown是什么

        Markdown 是一种轻量级的「标记语言」，它的常用标记符号不超过十个，学习成本低，一旦熟悉其语法规则，运用起来你的写作效率将得到质的提升，让我们更专注于码字，而基本的「标记」语法也让我们能应付日常的工作，同时Markdown支持多种格式的导出，如HTML、PDF，说了这么多，赶紧开始认识这门新语言吧，你一定会爱上它的！

一、为什么是Markdown

  • 入门门槛低
  • 专注内容而不是样式
  • 多种格式自由导出
  • 兼容所有文本编辑器

二、我该用什么编辑器？

个人只推荐一款软件：Sublime Text 3 以及插件Markdown Preview 和 MarkdownEditing

三、Markdown基本语法

字体

  • 斜体
    语法说明：*[文本]*
    说明：用两个*包涵一段文本，⚠️ 标记符号之间不能有空格

  • 粗体
    语法说明：**[文本]**
    说明：用两个**包涵一段文本，⚠️ 标记符号之间不能有空格

标题

语法规则：# [文本]
说明：#的个数代表标题的级数，级数越大，标题字号越小，一共有六级标题，⚠️ 要和文本之间加上一个字符的空格

列表

  • 有序列表
    语法规则： [数字]. [文本]
    说明: 直接在文本前加 1. 2. 3. 符号，⚠️ 要和文本之间加上一个字符的空格

  • 无序列表
    语法规则： [* 或 - 或 +] [文本]
    说明：* ,-,+ 3个符号效果都一样，这3个符号被称为Markdown符号，⚠️ 要和文本之间加上一个字符的空格

⚠️ 如果需要用到嵌套列表，请在子列表标记符号前缩紧2个空格

引用

语法规则： > 文本
说明：⚠️ 要和文本之间加上一个字符的空格

文本链接

语法规则： [锚文本] (链接地址)
说明：⚠️ 标记符号之间不能有空格

插入图片

语法规则： ![锚文本] (链接地址)
说明：插入图片和文本链接最大的区别就是有没有!号，⚠️ 标记符号之间不能有空格

表格

语法规则：

说明：插入图片和文本链接最大的区别就是有没有!号，⚠️ 标记符号之间不能有空格

代码块高亮

语法规则： ` [代码] `
说明：用两个 ` 把中间的代码包裹起来，就可以在里面写代码啦，可以使用tab进行缩进，⚠️ 要和文本之间加上一个字符的空格

⚠️ 你还可以通过以下方式为代码段着色,要在bash前加上一个字符的空格

``` bash
[代码段]
```

分割线

语法规则：*** 或 ---
说明：分割线的语法只需要另起一行，连续输入三个星号或减号即可

特殊符号

语法规则： \特殊符号
说明：用\来转义，输出的Markdown特殊符号,如# ，⚠️ \与标记符号之间不能有空格
